# Paz Invest

Projeto em Excel para controle e simulação de investimentos pessoais.

A planilha organiza aportes mensais, taxa de rendimento e período de investimento, permitindo simular o patrimônio acumulado ao longo do tempo.

## Ferramentas utilizadas
- Função VF (Valor Futuro)
- Fórmulas básicas (soma, procv e porcentagem)
- Simulação de cenários
- Organização dos dados em abas (Base, Resumo e Simulador, Validação de Dados)

## Observação
Os cálculos consideram a taxa real de rendimento, mesmo quando o Excel exibe valores arredondados.

## Objetivo
Praticar análise de dados no Excel, lógica financeira e organização de informações de forma simples e funcional.

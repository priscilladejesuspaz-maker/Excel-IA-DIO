# DIO - Excel com Inteligência Artificial

## Paz Invest

Projeto em Excel para controle e simulação de investimentos pessoais, com foco em análise de dados à tomada de decisão.

A planilha permite definir o perfil do investidor, organizar aportes mensais, aplicar percentuais por tipo de FII e simular o patrimônio acumulado ao longo do tempo.

## Ferramentas utilizadas
- Função VF (Valor Futuro)
- Fórmulas básicas (soma, PROC V e porcentagem)
- Validação de dados para seleção de perfis
- Simulação de cenários
- Organização dos dados em abas (Base, Resumo e Simulador)

## Observação
Os cálculos consideram a taxa real de rendimento, mesmo quando o Excel exibe valores arredondados.

## Objetivo
Praticar análise de dados no Excel, lógica financeira e organização de informações de forma simples e funcional.
